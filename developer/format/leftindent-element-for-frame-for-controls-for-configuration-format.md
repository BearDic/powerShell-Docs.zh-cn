# <a name="leftindent-element-for-frame-for-controls-for-configuration-format"></a>LeftIndent Element for Frame for Controls for Configuration (Format)

指定左边距远离数据向右移的字符数。 定义可由格式设置文件中的所有视图的公共控件时，将都使用此元素。

```xml

```
