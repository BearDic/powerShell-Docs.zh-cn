# <a name="propertyname-element-for-expressionbinding-for-controls-for-view-format"></a>PropertyName Element for ExpressionBinding for Controls for View (Format)

指定控件显示其值的.NET 属性。 定义一个视图，可以使用的控件时，将使用此元素。

配置元素 （格式） ViewDefinitions 元素 （格式） 视图元素 （格式） 控件元素 （格式） 控制元素的控件的视图 （格式） CustomEntries 元素的控件的控件的视图 （格式） CustomControl 元素CustomControl CustomEntries CustomEntry CustomItem 视图 （格式） 属性名称的控件的视图 （格式） ExpressionBinding 元素的控件的视图 （格式） CustomItem 元素的控件的视图 （格式） CustomEntry 元素ExpressionBinding 的视图 （格式） 的控件的元素
