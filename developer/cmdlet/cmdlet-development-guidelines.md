# <a name="cmdlet-development-guidelines"></a>Cmdlet 开发指南

在本部分中的主题提供可用于生成格式正确的 cmdlet 的开发指南。 通过利用提供通过 Windows PowerShell 运行时并遵循以下准则的常见功能，可以开发可靠 cmdlet 最小的工作量，并向用户提供一致的体验。 此外，您将降低测试负担，因为常见功能不需要重新测试。
