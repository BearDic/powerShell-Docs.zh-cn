# 声明基类
可以将 Windows PowerShell 类声明为另一个 Windows PowerShell 类的基类型。

```PowerShell
class bar
{
   [int]foo() 
       {
           return 100500
       }
}

class baz : bar {}

[baz]::new().foo() # return 100500
```

此外，还可以使用现有的.NET Framework 类型作为基类：

```PowerShell
class MyIntList : system.collections.generic.list[int]
{
    
}

$list = [MyIntList]::new()

$list.Add(100)

$list[0] # return 100
```
