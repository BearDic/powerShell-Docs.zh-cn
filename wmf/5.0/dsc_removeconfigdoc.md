# 删除 DSC 文档

将配置文档传送到 DSC 时，文档将经历各个阶段（挂起、当前、上一个）。 我们在 Windows PowerShell 4.0 中将新的 cmdlet Remove-DscConfigurationDocument 添加到了 DSC 作为 [KB3000850](https://support.microsoft.com/en-us/kb/3000850) 的一部分。
