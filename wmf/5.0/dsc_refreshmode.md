# <a name="additional-value-for-refreshmode-property"></a>RefreshMode 属性的附加值

此版本引入了新的 `RefreshMode` 值 **Disabled**。 设置为此模式时，LCM 不进行文档管理。 有关详细信息，请参阅[配置本地配置管理器](https://msdn.microsoft.com/powershell/dsc/metaconfig)。
